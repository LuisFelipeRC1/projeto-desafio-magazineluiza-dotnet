# 📦 Desafio Magazine Luiza – API de Agendamentos

API REST desenvolvida em **.NET** para gerenciamento de agendamentos de envio de mensagens.

O sistema permite criar, consultar e remover agendamentos, aplicando regras de domínio para garantir a consistência dos dados.

O projeto foi estruturado seguindo princípios de **Clean Architecture**, separando responsabilidades entre domínio, aplicação e infraestrutura.

---

# 🚀 Funcionalidades

- Criar um novo agendamento de envio de mensagem
- Consultar um agendamento pelo **ID**
- Remover um agendamento
- Controle de **status do agendamento**
- Validação de regras de negócio no domínio

---

# 🏗️ Arquitetura do Projeto

O projeto foi organizado em camadas para garantir **baixo acoplamento e alta coesão**.
